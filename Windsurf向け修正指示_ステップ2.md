# Windsurf向け修正指示プロンプト - UI設計書（ステップ2）

## 確認結果

UI設計書（ステップ2）の内容を詳細設計書と照合しました。
全体的に良好ですが、以下の点について修正・追加が必要です。

---

## 修正が必要な項目

### 1. 画面レイアウト設計.md - SC-10 期間予算編集画面の詳細化

**問題点**:
現在の記載では、Excel型UIの「月が横に並ぶ」という表現が抽象的です。
詳細設計書（`04_画面仕様と画面遷移.md` 293-303行目）には、以下の5行が横並びで表示されると明記されています。

**修正内容**:
`画面レイアウト設計.md` の 3.9 SC-10 期間予算編集画面（109-121行目）を以下のように修正してください。

**修正前**:
```
- 中央: Excel型UIテーブル（横スクロール）
  - 各月を横方向に並べたグリッド
  - 行: 年月 / 予算数量 / 月別売上 / 月別原価 / 月別粗利
```

**修正後**:
```
- 中央: Excel型UIテーブル（横スクロール）
  - 各月を横方向に並べたグリッド（例: 2025-01, 2025-02, 2025-03...）
  - 縦方向の行構成:
    - 行1: 年月（表示のみ）
    - 行2: 予算数量（編集可）
    - 行3: 月別売上（税別）（自動計算、表示のみ）
    - 行4: 月別原価（自動計算、表示のみ）
    - 行5: 月別粗利（自動計算、表示のみ）
  - 予算数量を変更すると、その列の月別売上・原価・粗利が即座に更新される
  - 横スクロールで期間内の全ての月を表示
```

---

### 2. 画面レイアウト設計.md - SC-01 トップページの詳細化

**問題点**:
「今月のサマリ」の表示内容が不明確です。
詳細設計書（`04_画面仕様と画面遷移.md` 76-81行目）には、売上・粗利・営業利益・各率が含まれると明記されています。

**修正内容**:
`画面レイアウト設計.md` の 3.1 SC-01 トップページ（27-40行目）を以下のように修正してください。

**修正前**:
```
- 上段: 今月のサマリ（マスター権限のみ表示）
  - 左: 売上・原価・粗利・広告費・営業利益（カード or ボックスで横並び）
  - 右: 率（原価率・粗利率・広告率・利益率）のサマリ
```

**修正後**:
```
- 上段: 今月のサマリ（マスター権限のみ表示）
  - 対象月: 現在月（YYYY-MM）を表示
  - 5つのカードを横並びで表示:
    1. 売上（税別）
    2. 粗利
    3. 営業利益（粗利 - 広告費）
    4. 原価率・粗利率
    5. 広告率・利益率
  - 各カードには金額と率（%）を併記
  - スタッフ権限の場合、このセクション全体を非表示
```

### 3. コンポーネント設計.md - データテーブルの詳細化

**問題点**:
データテーブルの「ソート」機能について、詳細設計書には「全列でソート可能」と明記されていますが、UI設計書では具体的な実装方法が記載されていません。

**修正内容**:
`コンポーネント設計.md` の 3.4 データテーブル（86-101行目）に以下を追加してください。

**追加位置**: 「共通機能」セクションの「ソート」の後

**追加内容**:
```
ソートの実装詳細:
- ヘッダーセルにソートアイコンを表示（▲▼）
- 初期状態: ソート未適用（アイコンはグレー）
- クリック1回目: 昇順ソート（▲が強調）
- クリック2回目: 降順ソート（▼が強調）
- クリック3回目: ソート解除（初期状態に戻る）
- 現在ソート中の列はヘッダー背景を薄く強調
```

### 4. コンポーネント設計.md - 期間予算編集画面の履歴モーダル

**問題点**:
詳細設計書（`04_画面仕様と画面遷移.md` 321-325行目）には、期間予算の履歴表示機能が明記されていますが、UI設計書のモーダルコンポーネントには記載がありません。

**修正内容**:
`コンポーネント設計.md` の 3.5 モーダル / ダイアログ（102-120行目）を以下のように修正してください。

**修正前**:
```
代表用途:

- 期間予算の履歴表示（SC-10）
- 売上CSV上書きモード時の確認
```

**修正後**:
```
代表用途:

- 期間予算の履歴表示（SC-10）
  - タイトル: 「期間予算の変更履歴」
  - ボディ: 履歴テーブル（保存日時、期間合計数量、月別内訳をJSON形式で表示）
  - フッター: 「閉じる」ボタン
- 売上CSV上書きモード時の確認
  - タイトル: 「確認」
  - ボディ: 「対象月（YYYY-MM）の全実績データを削除して上書きします。よろしいですか？」
  - フッター: 「OK」「キャンセル」ボタン
```

**修正内容**:

**追加内容**:
```
### 6.4 達成率の色分け

予算vs実績画面（SC-11）での達成率表示に使用:

- 100%以上: `#16A34A`（緑色）
  - 用途: 予算達成
- 80%以上100%未満: `#F97316`（黄色）

- または、テキスト色を変更
```
