### Windsurf UI 実装エージェント用プロンプト

以下の内容をコピーして Windsurf のチャット欄に貼り付けてください。

---

#### 📌 システム全体概要
- **プロジェクト**: Rinori 売上管理システム
- **スタック**: Next.js (App Router), Tailwind CSS, Prisma, SQLite
- **デザインコンセプト**: プロフェッショナルで「プレミアム感」のある洗練されたビジネスツール。
- **ベースカラー**: Rinori ネイビー (#00214d) と Rinori ゴールド (#d4af37) を基調としつつ、清潔感のある白と微細なグラデーションを活用。

#### 🎯 今回のミッション: V1.52 UI/UX 改善
バックエンドのロジック実装（除外キーワード判定、期間計算、ソートアルゴリズム等）は完了しています。
Windsurf には、これらの新機能を「直感的で美しいUI」に昇華させる実装を依頼します。

#### 🛠 具体的な実装タスク

**1. グローバルヘッダー & ナビゲーションのリファイン**
- [ ] ヘッダーのデザインをよりモダンに（シャドウ、透過背景、微細なアニメーション）。
- [ ] ナビゲーション項目を整理し、分析系（商品予算vs商品実績、PL分析）を優先表示。
- [ ] 「設定」リンクを追加。

**2. ダッシュボード (SC-01) のアップグレード**
- [ ] 数字カード（売上、利益率等）にアイコンやトレンド情報を追加し、視覚的に分かりやすく。
- [ ] 「新商品候補」や「不完全マスタ」の通知を、単なる警告ではなく「スマートなアクション通知」としてデザイン。

**3. 設定ハブ画面 (SC-??) の新規作成**
- [ ] `/settings/page.tsx` を、項目が整理されたグリッド形式またはリスト形式のモダンなメニュー画面に。
- [ ] 各設定項目（アカウント、販路、除外キーワード、ユーザー管理、税率等）に説明文とアイコンを付与。

**4. 除外キーワード管理画面 (新規) のデザイン**
- [ ] `/settings/exclusion-keywords/page.tsx` を、スッキリとしたリスト形式に。
- [ ] 追加フォームは入力しやすいフローティングまたはモーダル、あるいはページ上部に統合。

**5. 新商品候補一覧 (SC-06) のタブ・モーダル改善**
- [ ] 「未登録」と「無視リスト」の切り替えタブをよりスタイリッシュに。
- [ ] 一括登録モーダルのレイアウトを調整し、入力負荷を軽減。

**6. PL分析・予算管理テーブルの改善**
- [ ] `/pl/products` および `/budget/vs-actual` のテーブルに、ソート可能なヘッダー（アイコン付き）と、達成率に応じたプログレスバーやバッジを表示。
- [ ] `PeriodNavigator`（期間移動コンポーネント）のボタンを、押しやすく洗練されたデザインに共通化。

#### 💎 デザイン要求事項
- **Rich Aesthetics**: プレーンな Tailwind のデフォルト色ではなく、調整された HSL パレットを使用。
- **Interactive Elements**: ホバー時の色変化、スケール、フェードインアニメーションを多用し、動的な操作感を提供。
- **Clean Structure**: 情報密度が高くても圧迫感のない、余裕を持った余白（Spacing）の活用。
