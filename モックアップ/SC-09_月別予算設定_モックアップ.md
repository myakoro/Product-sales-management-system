# SC-09 月別予算設定画面 モックアップ

## 1. 画面概要
- 対象年月を指定し、商品×月の予算数量・金額を設定する画面。

## 2. レイアウト構成

1. ページタイトル
   - 「月別予算設定」

2. 対象年月選択エリア
   - 対象年月（YYYY-MM）ドロップダウン
   - 右側に「読込」ボタン

3. 商品別予算テーブル
   - 横幅フル、必要に応じて横スクロール
   - 列: 商品コード / 商品名 / 販売価格（税別） / 原価（税別） / 予算数量（編集可） / 予算売上（税別）（編集可） / 予算原価（税別）（自動） / 予算粗利（自動）

4. フッター操作
   - 右下に「保存」ボタン。

## 3. バリデーション
- 必須: 予算数量は0以上の整数。
- エラーはテーブル上部にアラート表示。
