# SC-10 期間予算編集画面 モックアップ

## 1. 画面概要
- 商品×期間の合計数量を入力し、期間内各月へ等分配・編集するExcel型UI画面。

## 2. レイアウト構成

1. ページタイトル
   - 「期間予算編集」

2. 期間情報フォーム
   - 商品コード選択（ドロップダウン）
   - 開始年月 / 終了年月（YYYY-MM）
   - 期間合計数量
   - 備考

3. 月別Excel型テーブル
   - 横方向に月（例: 2025-01, 2025-02, 2025-03 ...）を並べる。
   - 縦方向5行構成:
     - 行1: 年月（表示のみ）
     - 行2: 予算数量（編集可）
     - 行3: 月別売上（税別）（自動計算、表示のみ）
     - 行4: 月別原価（自動計算、表示のみ）
     - 行5: 月別粗利（自動計算、表示のみ）
   - 予算数量変更時、その列の売上・原価・粗利が即時更新される。
   - 横スクロールで期間内全月を表示。

4. 期間合計とズレ表示
   - 上部またはテーブル下部にサマリ領域:
     - 期間合計数量（固定値）
     - 現在の月別合計数量
     - ズレ（+3 / -2 など）
   - ズレが0でない場合、警告バナーを表示。

5. 操作ボタン
   - 右下: 「保存」ボタン。
   - ズレがある状態で保存しようとした場合、確認ダイアログを表示。
   - 「履歴を見る」ボタンで期間予算履歴モーダルを表示。

## 3. 履歴モーダル（概要）
- タイトル: 「期間予算の変更履歴」
- ボディ: 履歴テーブル（保存日時、期間合計数量、月別内訳JSON）
- フッター: 「閉じる」ボタン。
