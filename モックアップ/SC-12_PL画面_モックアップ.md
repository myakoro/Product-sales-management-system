# SC-12 PL画面（月次・期間） モックアップ

## 1. 画面概要
- 月次または任意期間のPL（損益）を表示する画面。権限により総合計の表示が変わる。

## 2. レイアウト構成

1. ページタイトル
   - 「PL（月次・期間）」

2. 期間選択エリア
   - 期間種別: 単月 / 3ヶ月 / 6ヶ月 / 期首〜現在 / 任意範囲
   - 任意範囲選択時は開始月・終了月入力
   - 「表示」ボタン

3. PLサマリボックス
   - マスター権限:
     - ボックス内レイアウト（縦並び）:
       - 売上（税別）
       - 原価（税別） + 原価率
       - 粗利 + 粗利率
       - 広告費 + 広告率
       - 営業利益 + 利益率
   - スタッフ権限:
     - 上記総合計ボックスは非表示
     - 「総合計はマスター権限のみ表示されます。」といった案内メッセージのみ表示。

## 3. 将来拡張
- 商品別PLリンクやグラフ表示は本画面からは行わず、別画面（SC-13）で対応。
