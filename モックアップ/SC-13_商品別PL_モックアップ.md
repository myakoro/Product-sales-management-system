# SC-13 商品別PL画面 モックアップ

## 1. 画面概要
- 指定期間における商品別PL（売上・原価・粗利・率）を一覧表示する画面。

## 2. レイアウト構成

1. ページタイトル
   - 「商品別PL」

2. 期間選択エリア
   - SC-11 と同様の期間指定UI（単月 / 複数月 / 3ヶ月 / 6ヶ月 / 期首〜現在）
   - 「表示」ボタン

3. 商品別PLテーブル
   - 列: 商品コード / 商品名 / 売上（税別） / 原価（税別） / 粗利 / 原価率 / 粗利率
   - ソート・フィルタ・ページングはデータテーブル共通仕様に従う。

## 3. 権限制御
- マスター・スタッフともに同一表示（総合計ではなく商品別のため）。
