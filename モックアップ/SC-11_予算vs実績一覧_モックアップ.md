# SC-11 予算 vs 実績一覧画面 モックアップ

## 1. 画面概要
- 期間を指定し、商品別の予算と実績の進捗を一覧で確認する画面。

## 2. レイアウト構成

1. ページタイトル
   - 「予算 vs 実績」

2. 期間選択エリア
   - 期間種別ラジオ: 単月 / 複数月範囲 / 3ヶ月 / 6ヶ月 / 期首〜現在
   - 必要に応じて開始月・終了月入力
   - 「表示」ボタン

3. 商品別サマリテーブル
   - 列: 商品コード / 商品名 / 予算数量 / 実績数量 / 数量達成率 / 予算売上（税別） / 実績売上（税別） / 売上達成率
   - 達成率セルにデザインシステムの色分けルールを適用。

## 3. ソート・フィルタ
- テーブルヘッダーでソート可能（ソート仕様はコンポーネント設計に準拠）。
