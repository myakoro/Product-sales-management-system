# SC-05 商品マスタ一覧画面 モックアップ

## 1. 画面概要
- 商品マスタの一覧表示・検索・フィルタ・編集遷移を行う画面。

## 2. レイアウト構成

1. ページタイトル
   - 「商品マスタ一覧」

2. フィルタ・検索エリア
   - 行1: 商品区分セレクト（自社/仕入）、管理ステータスセレクト（管理中/管理外）
   - 行2: キーワード検索（商品コード・商品名）
   - 右上: 「新規登録」ボタン

3. 商品一覧テーブル
   - 列: 商品コード / 商品名 / 販売価格（税別） / 原価（税別） / 商品区分 / 管理ステータス / 登録日 / 操作（編集）
   - 行末に「編集」ボタン。

## 3. アラート
- 不完全マスタの警告は主にトップページで扱うため、本画面ではテーブル上部に補足テキスト程度に留める。
