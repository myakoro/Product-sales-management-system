# SC-01 トップページ / ダッシュボード モックアップ

## 1. 画面概要
- システムの入口画面。
- 今月のサマリ、主要商品の予算 vs 実績、警告・通知、各機能へのナビゲーションを表示。

## 2. レイアウト構成

### 2.1 ヘッダー
- 左: システム名「Rinori 販売管理システム」
- 中央〜右: ナビゲーションメニュー
- 右端: ログインユーザー名 + 権限表示、ログアウトボタン

### 2.2 メインコンテンツ

1. 今月のサマリ（マスター権限のみ表示）
   - タイトル: 「今月のサマリ（YYYY-MM）」
   - 横1列に5つのカードを並べる:
     1. 売上（税別）
     2. 粗利
     3. 営業利益（粗利 - 広告費）
     4. 原価率・粗利率
     5. 広告率・利益率
   - 各カード:
     - 上段: ラベル（例: 「売上（税別）」）
     - 中段: 金額（大きめの数字）
     - 下段: 率（%）

2. 今月の主要商品 予算 vs 実績
   - タイトル: 「今月の主要商品 予算 vs 実績」
   - テーブル（上位5商品）
     - 列: 商品コード / 商品名 / 予算数量 / 実績数量 / 達成率
     - 達成率セルはデザインシステムの色分けルールに従う。

3. 警告・通知エリア
   - 左側に縦積みで表示:
     - 不完全マスタ警告（件数 + 詳細へのリンク）
     - 新商品候補の件数 + 一覧画面へのリンク

4. ショートカットリンク
   - 主要機能へのボタンをグリッド状に配置:
     - 売上CSV取込
     - 商品マスタ一覧
     - 月別予算設定
     - 期間予算編集
     - 予算 vs 実績
     - PL画面
     - 商品別PL
     - 広告費一覧
     - 税率設定

### 2.3 フッター
- 左: コピーライト
- 右: バージョン情報

## 3. 権限制御
- マスター権限: すべてのセクションを表示。
- スタッフ権限:
  - 「今月のサマリ」セクションは非表示。
  - 主要商品 予算 vs 実績、警告・通知、ショートカットリンクは表示。
