# SC-07 新商品候補一覧画面 モックアップ

## 1. 画面概要
- 売上CSVから検出された新商品候補を一覧し、管理中/管理外として登録または無視を選択する画面。

## 2. レイアウト構成

1. ページタイトル
   - 「新商品候補一覧」

2. 操作ボタンエリア
   - テーブル上部に配置:
     - 「選択した候補を管理中として登録」
     - 「選択した候補を管理外として登録」
     - 「選択した候補を無視」

3. 候補一覧テーブル
   - 列: チェックボックス / 親コード / 元SKU例 / 商品名 / 検出日時 / ステータス

## 3. メッセージ
- 候補が0件の場合、「新商品候補はありません」とセンタリングメッセージを表示。
