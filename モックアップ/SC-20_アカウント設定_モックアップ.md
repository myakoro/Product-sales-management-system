# SC-20 アカウント設定モックアップ

## 画面概要

- 画面名: アカウント設定
- 目的: ログイン中ユーザーが自分のパスワードを変更する。
- 対象: マスター権限 / スタッフ権限 共通（ログイン必須）

## レイアウト

- ヘッダー: 既存画面と同じ共通ヘッダー（システム名 / ナビ / ユーザー情報 / ログアウト）
- 本文:
  - ページタイトル: 「アカウント設定」
  - セクション: 「パスワード変更」
    - 説明文: 機能の概要を1〜2行で表示
    - 入力フォーム（縦並び）
      - 現在のパスワード（password）
      - 新しいパスワード（password）
      - 新しいパスワード（確認）（password）
      - 「パスワードを変更」ボタン

## 想定動作（モック段階）

- ボタン押下時の実処理は未実装。
- 将来的には以下の動作を想定:
  - 現在のパスワード検証
  - 新パスワードの強度・一致チェック
  - 成功時のメッセージ表示と再ログイン誘導

## 備考

- 詳細なバリデーションやエラーメッセージの内容は、第5部「処理フローとバリデーション」で定義予定。
- 本モックアップはレイアウト確認と項目漏れ防止が目的。
