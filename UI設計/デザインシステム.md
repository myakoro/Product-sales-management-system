# Rinori 販売管理システム - デザインシステム

## 1. 概要
Rinori 販売管理システムの画面全体で共通して利用する、色・タイポグラフィ・スペーシング・コンポーネントの基本ルールを定義する。

対象:
- SC-01〜SC-15, SC-18 の全画面
- 権限制御（マスター/スタッフ）を含む共通レイアウト

- ベースフォント: システムUIフォント（例: "system-ui", "-apple-system", "BlinkMacSystemFont", "Segoe UI", sans-serif）

### 3.2 フォントサイズ
- ページタイトル: 20〜24px

- テーブルヘッダー: 14〜16px
- 本文テキスト: 13〜14px

## 4. スペーシング

- ページ余白: 左右24px程度
- セクション間: 16〜24px
- テーブル行間: 8〜12px
- フォーム項目間: 8〜12px

## 6. カラーパレット詳細

### 6.1 ベースカラー（確定案）

- プライマリカラー: `#2563EB`  
  - 用途: 主要ボタン、強調リンク、アクティブ状態のタブ
- セカンダリカラー: `#4B5563`  
  - 用途: サブボタン、サブタイトル、補助テキスト
- アクセントカラー: `#E11D48`  
  - 用途: 重要な警告や重要指標の強調
- 背景色（ページ全体）: `#F9FAFB`  
  - 用途: 画面全体の背景
- カード / パネル背景: `#FFFFFF`  
  - 用途: コンテンツ領域（ボックス）
- 枠線・区切り線: `#E5E7EB`  
  - 用途: テーブル罫線、フォームの境界線、カードの境界線

### 6.2 状態カラー（確定案）

- 成功: `#16A34A`  
  - 用途: 成功メッセージ、成功バッジ
- 警告: `#F97316`  
  - 用途: 期間予算のズレ警告、注意メッセージ
- エラー: `#DC2626`  
  - 用途: バリデーションエラー、致命的エラー
- 情報: `#0EA5E9`  
  - 用途: 補足情報、ヘルプテキスト

### 6.3 テキストカラー

- メインテキスト: `#111827`  
- サブテキスト: `#4B5563`  
- 無効状態テキスト: `#9CA3AF`

### 6.4 達成率の色分け

予算vs実績画面（SC-11）での達成率表示に使用:

- 100%以上: `#16A34A`（緑色）
  - 用途: 予算達成
- 80%以上100%未満: `#F97316`（黄色）
  - 用途: 予算未達だが許容範囲
- 80%未満: `#DC2626`（赤色）
  - 用途: 予算大幅未達

実装方法:

- テーブルセルの背景色を薄く着色
- または、テキスト色を変更

## 9. コンポーネント共通スタイル方針（概要）

### 9.1 ボタン

- 角丸: 4px
- 高さ: 32〜36px
- フォントサイズ: 14px
- プライマリボタン: プライマリカラー背景 + 白文字
- セカンダリボタン: 白背景 + 枠線（プライマリカラー）+ プライマリ文字色
- 無効状態: 背景 `#E5E7EB`、テキスト `#9CA3AF`

### 9.2 入力フォーム

- テキストフィールド/セレクト/日付ピッカー共通:
  - 枠線: `#D1D5DB`
  - 角丸: 4px
  - フォーカス時: 枠線をプライマリカラーに変更

### 9.3 テーブル

- ヘッダー行背景: `#F3F4F6`
- 交互行: 必要に応じて `#F9FAFB` を利用
- 行ホバー時: 背景を薄いグレーに変更（例: `#F3F4F6`）

### 9.4 アラート / バナー

- 成功: 成功カラーの薄い背景 + 左側に色帯
- 警告: 警告カラーの薄い背景 + アイコン
- エラー: エラーカラーの薄い背景 + 強調テキスト

### 9.5 ローディング

- 画面全体ローディング: 半透明オーバーレイ + スピナー
- コンポーネント内ローディング: 行内スピナー or プレースホルダ表示
