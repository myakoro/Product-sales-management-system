# Rinori 販売管理システム - 画面レイアウト設計

## 1. 共通レイアウト方針

- 対象ブラウザ: PC版 Google Chrome（最小幅1280px想定）
- 共通要素:
  - ヘッダー: システム名、ナビゲーション、ユーザー名/権限、ログアウトボタン
  - メインコンテンツ: 画面ごとの内容
  - フッター: コピーライト、バージョン情報

## 2. 画面グループ別レイアウト

- SC-01: トップページ / ダッシュボード
- SC-02〜SC-04: CSV取込関連
- SC-05〜SC-07: 商品マスタ・新商品候補
- SC-09: 予算設定（統合）
- SC-11〜SC-13: PL・分析
- SC-14: 広告費管理（統合）
- SC-18: 税率設定

各画面の詳細レイアウトは `04_画面仕様と画面遷移.md` に基づき、本ファイル内でテキストベースのワイヤーフレームとして定義する。

---

## 3. 画面別レイアウト詳細（ワイヤーフレーム）

### 3.1 SC-01 トップページ / ダッシュボード

構成:

- 上段: 今月のサマリ（マスター権限のみ表示）
  - 対象月: 現在月（YYYY-MM）を表示
  - 5つのカードを横並びで表示:
    1. 売上（税別）
    2. 粗利
    3. 営業利益（粗利 - 広告費）
    4. 原価率・粗利率
    5. 広告率・利益率
  - 各カードには金額と率（%）を併記
  - スタッフ権限の場合、このセクション全体を非表示
- 中段: 今月の主要商品 予算 vs 実績（テーブル）
  - 列: 商品コード / 商品名 / 予算数量 / 実績数量 / 達成率
- 下段: 警告・通知エリア + ショートカットリンク
  - 左: 不完全マスタ警告、新商品候補数
  - 右: 各機能へのショートカットボタン

ヘッダー直下にページタイトル「トップページ / ダッシュボード」を配置し、縦方向にセクションを積み重ねる 1カラム構成。

### 3.2 SC-02 売上CSV取込画面

構成:

- 上部フォームブロック（カード）
  - 1行目: 対象年月（ドロップダウン or テキスト）
  - 2行目: 取込モード（ラジオボタン: 追加/上書き）
  - 3行目: コメント（テキストエリア）
  - 4行目: CSVファイル選択 + 「取込実行」ボタン（右寄せ）
- 下部: 取込結果のメッセージエリア
  - 成功/エラー/警告メッセージ
  - 新商品候補検出件数などの要約

ページ全体は 1カラムで、中央寄せのフォームカードを配置。

### 3.3 SC-03 商品一覧CSV取込画面

構成:

- 上部フォームブロック
  - CSVファイル選択 + コメント + 「差分チェック」ボタン
- 下部: 差分候補テーブル（チェックボックス付き）
  - テーブル上部に「登録・更新」ボタンを配置

### 3.4 SC-04 CSV取込履歴画面

構成:

- 上部: フィルタ（期間、取込種別）
- 下部: 履歴テーブル（ページング付き）

### 3.5 SC-05 商品マスタ一覧画面

構成:

- 上部: フィルタ・検索エリア
  - 行1: 商品区分（自社/仕入）、管理ステータス（管理中/管理外）
  - 行2: キーワード検索（商品コード・商品名）
  - 右上: 「新規登録」ボタン
- 下部: 商品マスタ一覧テーブル
  - 最右列に「編集」ボタン

### 3.6 SC-06 商品マスタ編集画面

構成:

- 中央寄せフォームカード
  - 左1カラムで上から順に: 商品コード / 商品名 / 販売価格 / 原価 / 商品区分 / 管理ステータス
  - 下部右寄せに「保存」「キャンセル」ボタン

### 3.7 SC-07 新商品候補一覧画面

構成:

- 上部: 説明テキスト + 操作用ボタン群（「管理中として登録」「管理外として登録」「無視」）
- 下部: 候補一覧テーブル（チェックボックス付き）

### 3.8 SC-09 予算設定画面（統合）

構成:

- 上部フォームブロック
  - 行1: 期間選択（開始年月〜終了年月）+ 「表示」ボタン
  - 行2: 期間合計サマリ（売上 / 粗利 / 粗利率）
- 中央: Excel型UIテーブル（横スクロール）
  - 固定列（左）: 商品コード / 商品名 / 期間合計数量 / 期間売上 / 期間粗利
  - スクロール列（右）: 月別数量（01月, 02月...）
  - 期間合計数量または月別数量を変更すると、即座に全項目が再計算される
- 下部右寄せ: 「保存」ボタン

### 3.10 SC-11 予算 vs 実績一覧画面

構成:

- 上部: 期間選択（単月/複数月/3ヶ月/6ヶ月/期首〜現在）
- 中央: 商品別サマリテーブル

### 3.11 SC-12 PL画面（月次・期間）【V1.3 UI改善】

構成:

- 上部: 期間選択（ラジオボタン方式）
  - ◉ プリセット期間：期間種別（単月/3ヶ月/6ヶ月/期首〜現在）
  - ○ 任意の期間：開始年月〜終了年月
  - 選択した方式のみが有効、非選択はグレーアウト
- 中央: PLサマリボックス
  - マスター権限: 売上/原価/粗利/広告費/営業利益を1つのカード群で表示
  - スタッフ権限: 総合計部分を非表示（代わりに案内メッセージ）

### 3.12 SC-13 商品別PL画面

構成:

- 上部: 期間選択
- 中央: 商品別PLテーブル

### 3.14 SC-14 広告費管理画面（統合・UX改善）

構成（タブ方式）:

**タブ1: 広告費一覧**
- 上部: 期間フィルタ、カテゴリフィルタ
- 中央: 広告費一覧テーブル
  - 列: 日付 / 金額 / カテゴリ / メモ / 操作
  - 既存行はインライン編集可能
- 下部: 新規追加行（テーブル最下部）
  - 常に表示される入力行
  - 「保存」ボタンで行追加

**タブ2: カテゴリ管理**
- 中央: カテゴリ一覧テーブル
  - 列: カテゴリ名 / 作成日 / 操作
  - インライン編集可能
- 下部: 新規追加行
  - カテゴリ名入力 + 「保存」ボタン

### 3.15 SC-18 税率設定画面

構成:

- 上部: 新規税率登録フォーム（適用開始年月 + 税率 + 「追加」ボタン）
- 下部: 税率一覧テーブル（編集/削除ボタン付き）
